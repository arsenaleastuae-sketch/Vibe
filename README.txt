# 360° панорама с 7 хот-спотами (коктейли)

## Что внутри
- `index.html` — страница
- `style.css` — стили (белая плашка, черный текст, пульсирующая белая точка)
- `app.js` — логика + координаты хот-спотов (yaw/pitch) и контент
- `assets/pano.webp` — ваша панорама
- `assets/cocktail-1.svg ... cocktail-7.svg` — заглушки картинок коктейлей (можно заменить на фото .jpg/.png)

## Как загрузить на сервер
1) Скопируйте всю папку на хостинг (с сохранением структуры).
2) Откройте в браузере: `https://ваш-домен/путь/index.html`

## Где менять данные
- Фото: замените `assets/cocktail-N.svg` на ваши файлы (или поменяйте пути в `app.js` -> поле `img`)
- Характеристики/названия/ссылки: `app.js` -> массив `HOTSPOTS`
- Действие кнопки "Заказать": обработчик `orderBtn` в `app.js`

## Примечание
Сейчас Marzipano подключен через CDN (unpkg). Если нужен полностью офлайн пакет (без интернета) — скажите, подготовлю версию с локальным `marzipano.js`.
